# Kenya CBE Lesson Plan Generation System

> **Generated:** 2026-05-31 | Kenya CBE Lesson Plan Generation System


Automated generation of ~2,000 Kenya Competency-Based Education (CBE) lesson plans as formatted Word documents, covering Grade 10 Biology, Physics, Chemistry, and Mathematics.

Lesson plans are aligned to the official KICD (Kenya Institute of Curriculum Development) March 2025 curriculum and follow a phenomenon-driven, inquiry-based pedagogy (NGSS Storyline model). They are designed to be hosted on ARES Education offline servers (Rachel 4 Plus devices) in Kenyan schools.

---

## Quick Start

```bash
# Activate environment
cd /home/markk/ares/cbe-generation-system
source venv/bin/activate

# Generate a new sub-strand (synchronous)
python3 src/generate_substrand.py \
  --subject biology \
  --substrand 1.4 \
  --output bio_1_4 \
  --lessons 6 \
  --run

# Generate a new sub-strand (batch API — 50% cheaper, async)
python3 src/generate_substrand.py \
  --subject biology \
  --substrand 1.4 \
  --output bio_1_4 \
  --lessons 6 \
  --batch

# Collect batch results when ready
python3 src/generate_substrand.py --collect bio_1_4 --wait --run

# Regenerate docx from existing data file
node generators/generate.js bio_1_4

# Regenerate all sub-strands
node generators/generate.js --all
```

**Server:** jhm-spark (NVIDIA Spark, GB10 Blackwell GPU, 128GB RAM)
**OS:** Ubuntu 24.04

## Each Sub-Strand Produces Three Documents

| File | Description |
|---|---|
| `*_CBE_LessonSequence.docx` | Full lesson sequence (8+ lessons) for classroom use |
| `*_FinalExplanation.docx` | Student assessment document with exemplar answers |
| `*_SummaryTable.docx` | Teacher reference table (pre-filled summary per lesson) |

## Key Design Decisions

- **Source of truth is the data file.** Each `generators/data/*_data.js` file is the editable content artifact. Both docx outputs are generated from it — edit the data file, regenerate both.
- **ARES resources are auto-injected.** The Resource column in Section C is populated at generation time from the ARES content database (1.55M items) via FTS search. No manual resource lookup needed.
- **Batch API for scale.** Use `--batch` for any run of 3+ sub-strands. At 50% off standard pricing, the full 2,000-lesson target costs ~$114.
- **Claude API generates content; jhm-spark generates documents.** The API is stateless and handles curriculum design. jhm-spark holds all state, runs all formatting, and serves content to schools.
